# 计算机视觉小鼠管理系统 - 界面概览

## 系统简介

本系统是一个集成了计算机视觉、物联网和数据分析技术的智能化小鼠管理平台，基于RuoYi框架开发。系统通过红外摄像头和人工智能技术实现小鼠的自动识别、行为监测和数据分析，为科研人员提供从实验准备到数据报告的全流程自动化解决方案。

### 核心技术特色
- **计算机视觉识别**：自动识别小鼠个体和行为模式
- **二维码管理**：笼子智能标识和追踪系统
- **实时数据处理**：毫秒级行为数据采集和分析
- **智能报表生成**：自动化数据分析和多格式报告导出

## 主要界面展示

### 1. 系统登录界面

- **功能**：用户身份验证，确保数据安全
- **特点**：简洁的登录表单，支持记住密码功能
- **用户类型**：管理员、研究员、观察员等不同权限角色

### 2. 系统首页（智能仪表板）

- **实时监控面板**：
  - 当前活跃笼子数量和状态
  - 正在进行的实验数量
  - 系统设备在线状态（摄像头、传感器）
- **数据统计图表**：
  - 今日/本周/本月实验完成情况
  - 小鼠行为数据实时趋势
  - 异常行为检测和预警信息
  - 数据质量监控指标
- **快速操作入口**：
  - 扫描二维码激活笼子
  - 创建新实验
  - 查看实时监控
  - 生成分析报告
- **系统状态指示**：
  - 计算机视觉引擎状态
  - 数据处理队列状态
  - 存储空间使用情况

### 3. 笼子管理界面

#### 3.1 笼子注册与二维码管理
- **笼子注册**：
  - 一键生成笼子二维码标签
  - 笼子基本信息录入（位置、容量、类型）
  - 二维码打印和标签管理
- **笼子状态监控**：
  - 未激活、已激活、实验中、维护中等状态
  - 摄像头连接状态和图像质量检查
  - 环境参数监控（温度、湿度、光照）

#### 3.2 小鼠信息管理
- **扫码激活流程**：
  - 二维码扫描界面（支持手机和扫码枪）
  - 摄像头自动启动和小鼠检测
  - 实时显示检测到的小鼠数量和位置
- **小鼠信息确认**：
  - 自动识别结果展示
  - 人工信息补充和确认
  - 小鼠个体标记和特征记录
- **批量管理功能**：
  - 多笼子同时激活
  - 批量信息导入导出
  - 小鼠信息模板管理

### 4. 实验管理界面

#### 4.1 实验列表
- **实验状态**：准备中、进行中、已完成、已暂停
- **实验信息**：实验编号、开始时间、持续时间、负责人、参与笼子数
- **操作按钮**：查看详情、编辑参数、暂停/恢复、导出数据
- **快速筛选**：按状态、时间、负责人等条件筛选

#### 4.2 创建新实验
- **基本信息设置**：
  - 实验名称和详细描述
  - 实验类型（药物疗效、行为学测试、疾病模型等）
  - 预计持续时间和关键时间节点
- **笼子和小鼠选择**：
  - 从已激活笼子中选择参与实验的笼子
  - 自动关联笼子内的小鼠信息
  - 支持实验分组和对照组设置
- **监测参数配置**：
  - 数据采集频率（1-30帧/秒）
  - 重点监测行为类型选择
  - 异常行为预警阈值设定
  - 数据存储和备份策略

### 5. 智能监控与分析界面

#### 5.1 实时视觉监控
- **多路视频流**：
  - 同时显示多个笼子的实时画面
  - 支持画面缩放、全屏和画中画模式
  - 小鼠个体自动标记和追踪框显示
- **行为实时识别**：
  - 实时显示识别到的行为类型和置信度
  - 行为事件时间轴和统计计数器
  - 异常行为实时预警和高亮显示
- **轨迹可视化**：
  - 小鼠移动路径实时绘制
  - 活动热力图动态更新
  - 多只小鼠轨迹颜色区分

#### 5.2 智能数据分析
- **行为统计分析**：
  - 各类行为频次和持续时间统计
  - 昼夜节律分析和睡眠质量评估
  - 个体差异分析和群体行为模式
- **时间序列分析**：
  - 可自定义时间范围的数据趋势图
  - 支持小时、天、周、月等多时间尺度
  - 行为变化趋势预测和异常检测
- **对比分析功能**：
  - 实验组vs对照组数据对比
  - 给药前后行为变化分析
  - 多个实验项目横向对比
- **数据质量监控**：
  - 识别准确率实时监控
  - 数据完整性和一致性检查
  - 异常数据标记和处理建议

### 6. 智能报告生成界面

- **报告模板选择**：
  - 标准实验报告（包含基础统计和图表）
  - 详细分析报告（包含深度数据挖掘结果）
  - 对比分析报告（多组实验对比）
  - 自定义报告模板（用户可自定义内容和格式）
- **数据筛选与配置**：
  - 按时间范围、实验类型、小鼠个体等多维度筛选
  - 选择包含的行为指标和统计方法
  - 设置置信区间和显著性检验参数
- **图表和可视化定制**：
  - 行为趋势图、热力图、轨迹图等多种图表类型
  - 支持图表样式、颜色、标注的个性化设置
  - 3D可视化和交互式图表选项
- **多格式导出**：
  - PDF（适合打印和正式报告）
  - Excel（便于进一步数据分析）
  - Word（便于编辑和整合）
  - 原始数据CSV格式导出

#### 7.1 用户与权限管理

## 典型使用流程

### 研究人员日常操作流程
1. **登录系统** → 查看智能仪表板，了解当前实验状态
2. **笼子准备** → 扫描二维码激活笼子，确认小鼠信息
3. **创建实验** → 设置实验参数、监控时长和分析指标
4. **启动智能监控** → 系统自动开始计算机视觉识别和数据采集
5. **实时监控** → 查看AI识别结果、行为统计和异常预警
6. **数据分析** → 使用智能分析工具进行深度数据挖掘
7. **生成报告** → 选择报告模板，自动生成专业分析报告
8. **数据导出** → 导出原始数据或分析结果用于进一步研究

### 管理员维护操作流程
1. **系统状态检查** → 监控设备在线状态、AI引擎性能和数据质量
2. **用户权限管理** → 添加新用户、分配角色权限、审计操作日志
3. **设备维护管理** → 检查摄像头状态、更新AI模型、优化识别参数
4. **数据管理** → 执行数据备份、清理历史数据、监控存储空间
5. **系统优化** → 升级软件版本、调优算法参数、性能监控分析
6. **质量控制** → 检查数据完整性、处理异常数据、生成质量报告
