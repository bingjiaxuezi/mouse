# 计算机视觉小鼠管理系统 - 界面概览

## 系统简介

本系统是一个集成了计算机视觉、物联网和数据分析技术的智能化小鼠管理平台，基于RuoYi框架开发。系统通过红外摄像头和人工智能技术实现小鼠的自动识别、行为监测和数据分析，为科研人员提供从实验准备到数据报告的全流程自动化解决方案。

### 核心技术特色
- **计算机视觉识别**：自动识别小鼠个体和行为模式
- **二维码管理**：笼子智能标识和追踪系统
- **实时数据处理**：毫秒级行为数据采集和分析
- **智能报表生成**：自动化数据分析和多格式报告导出

## 主要界面展示

### 1. 系统登录界面

- **功能**：用户身份验证，确保数据安全
- **特点**：简洁的登录表单，支持记住密码功能
- **用户类型**：管理员、研究员、观察员等不同权限角色

### 2. 系统首页（实验驱动仪表板）

- **实验驱动仪表板**：
  - 当前进行中的实验项目状态（实验阶段、进度、剩余时间）
  - 实验二维码生成和管理中心
  - 今日实验数据采集量和AI分析进度
  - 系统健康状态和设备绑定情况
- **实验快速创建区**：
  - **一键实验创建**：选择实验模板，快速生成实验方案
  - **实验二维码生成**：自动生成包含实验参数的二维码
  - **设备扫码绑定**：显示待绑定设备数量和绑定状态
  - **实验模板库**：常用实验类型的预设模板
- **实验状态概览**：
  - **进行中实验**：显示各实验的实时状态和关键指标
  - **待启动实验**：已创建但未开始的实验列表
  - **AI分析结果**：最新的智能分析结果和异常预警
  - **设备使用情况**：摄像头绑定状态和数据采集效率

### 3. 实验绑定管理界面（原笼子管理）

#### 3.1 实验二维码扫描中心
- **扫码绑定功能**：
  - 摄像头扫描实验二维码，自动获取实验配置
  - 实验信息显示（实验名称、参数、监测要求等详细信息）
  - 设备自动配置（根据实验要求自动配置摄像头参数）
  - 绑定状态管理（显示已绑定、待绑定、异常等状态）
- **智能笼子识别**：
  - 自动笼子检测（摄像头自动识别笼子并生成笼子ID）
  - 笼子信息确认（确认笼子位置、尺寸、环境参数）
  - 实验关联（笼子自动绑定到当前扫描的实验项目）
  - 状态指示（显示"已就绪"、"等待小鼠"、"监测中"等状态）

#### 3.2 小鼠智能添加与实验启动
- **自动识别流程**：
  - 小鼠放入后自动检测数量和基本特征
  - 实时显示检测到的小鼠数量和位置
  - AI识别结果展示和置信度显示
- **信息确认与分组**：
  - 研究人员确认小鼠的品系、性别、年龄等信息
  - 根据实验设计自动分配到实验组或对照组
  - 小鼠个体标记和特征记录
- **监测启动管理**：
  - 小鼠信息确认后自动开始数据采集
  - 批量实验启动和状态同步
  - 实验进度跟踪和异常处理

### 4. 实验驱动管理中心

#### 4.1 实验前置创建与配置
- **实验方案设计**：
  - **实验模板选择**：药物疗效评价、疾病模型研究、昼夜节律等预设模板
  - **参数智能配置**：实验周期、监测行为、采集频率等一键设置
  - **预期结果设定**：设置预期的统计学指标和显著性水平
  - **实验二维码生成**：自动生成包含所有配置信息的实验二维码
- **智能实验设计**：
  - **样本量计算**：AI根据预期效应量自动计算所需样本量
  - **分组策略建议**：智能推荐最佳的实验分组方案
  - **时间节点规划**：自动生成实验时间表和关键节点提醒
  - **质量控制设置**：预设数据质量检查规则和异常处理策略

#### 4.2 实验全程智能管控
- **一键启动管理**：
  - **批量设备激活**：扫描实验码后所有相关设备同时启动
  - **实验状态同步**：所有笼子状态实时同步到实验管理中心
  - **进度可视化**：实验时间轴、完成度、剩余时间等直观显示
  - **阶段自动切换**：基线期→干预期→分析期的自动状态切换
- **智能监控与预警**：
  - **AI实时分析**：实验数据的实时统计分析和趋势预测
  - **异常智能识别**：AI自动识别数据异常和实验风险
  - **自动报警系统**：设备故障、数据异常、实验偏差的即时通知
  - **质量实时监控**：数据完整性、识别准确率的持续监控

### 5. 实验智能监控与AI分析中心

#### 5.1 实验驱动监控大屏
- **实验状态总览**：
  - **实验进度仪表板**：显示所有进行中实验的实时状态和完成度
  - **AI分析实时更新**：实验数据的实时统计分析和趋势预测
  - **异常实验预警**：AI自动识别实验偏差和数据异常
  - **质量监控指标**：数据完整性、识别准确率、设备状态等关键指标
- **智能视频监控**：
  - **实验关联显示**：每个视频窗口显示对应的实验信息和阶段
  - **AI识别增强**：小鼠行为的实时识别和统计学分析
  - **实验组对比**：实验组与对照组的行为数据实时对比
  - **关键事件捕获**：AI自动捕获和标记实验关键时刻

#### 5.2 实验数据智能可视化
- **实验轨迹分析**：
  - **分组轨迹对比**：实验组与对照组的运动轨迹对比分析
  - **药效时间曲线**：药物干预前后的行为变化趋势
  - **个体响应分析**：识别药物高响应者和低响应者
  - **统计学验证**：轨迹数据的统计显著性检验
- **实验行为时间线**：
  - **实验阶段标记**：基线期、干预期、恢复期的清晰划分
  - **药效动力学分析**：药物作用的起效时间、峰值效应、持续时间
  - **剂量-效应关系**：不同剂量组的效应对比分析

#### 5.3 AI驱动的实验分析
- **智能统计分析**：
  - **自动假设检验**：AI自动选择合适的统计学方法进行分析
  - **效应量计算**：自动计算Cohen's d、效应量置信区间等指标
  - **多重比较校正**：自动进行Bonferroni、FDR等多重比较校正
  - **统计功效分析**：实时评估实验的统计功效和样本量充分性
- **预测性分析**：
  - **药效预测模型**：基于早期数据预测最终实验结果
  - **个体化分析**：识别影响药效的个体特征和预测因子
  - **实验优化建议**：AI提供实验参数优化和改进建议

### 6. 实验驱动智能报告中心

#### 6.1 实验报告一键生成
- **实验模板智能匹配**：
  - **自动模板识别**：AI根据实验类型自动选择最适合的报告模板
  - **药物疗效评价报告**：包含药效动力学、统计学分析、安全性评价
  - **疾病模型验证报告**：包含模型成功率、表型特征、验证指标
  - **个体化医学报告**：包含个体差异分析、响应者识别、精准医学建议
- **AI智能报告生成**：
  - **实验全程自动记录**：从实验创建到数据分析的完整记录
  - **智能结论生成**：AI基于统计分析自动生成专业结论和建议
  - **多格式一键导出**：支持学术论文格式、监管申报格式、内部汇报格式
  - **实时报告更新**：实验进行中报告内容实时更新和优化

#### 6.2 专业级分析报告
- **深度统计分析**：
  - **自动统计学检验**：AI自动选择t检验、方差分析、非参数检验等方法
  - **效应量与置信区间**：自动计算Cohen's d、95%置信区间等专业指标
  - **功效分析与样本量**：评估实验统计功效和样本量充分性
  - **多重比较校正**：自动进行Bonferroni、FDR等校正方法
- **可视化与解读**：
  - **专业统计图表**：森林图、小提琴图、生存曲线等高级可视化
  - **AI智能解读**：对统计结果进行专业解读和临床意义分析
  - **实验质量评估**：数据质量、实验设计合理性的综合评估
  - **改进建议生成**：AI提供实验优化和后续研究建议

## 实验驱动典型使用流程

### 研究人员实验驱动操作流程
1. **实验前置设计** → 登录系统，选择实验模板或创建新实验方案
2. **实验参数配置** → 设置实验周期、监测行为、分组策略等参数
3. **生成实验二维码** → 系统自动生成包含实验配置的二维码
4. **扫码绑定设备** → 在实验室扫描实验二维码，自动绑定摄像头设备
5. **智能笼子识别** → 系统自动识别笼子并关联到实验项目
6. **小鼠智能添加** → 放入小鼠后系统自动识别，确认基本信息和分组
7. **一键启动实验** → 所有准备就绪后一键启动实验监测
8. **实验智能监控** → AI实时分析实验数据，自动预警异常情况
9. **实验结果分析** → 实验完成后AI自动进行统计分析
10. **一键生成报告** → 选择报告格式，AI自动生成专业实验报告

### 管理员实验系统维护流程
1. **实验系统监控** → 检查所有进行中实验的状态和数据质量
2. **实验设备管理** → 监控实验设备与项目的绑定状态
3. **实验权限管理** → 管理实验团队成员的协作权限
4. **实验数据备份** → 按实验项目进行数据备份和归档
5. **实验模板优化** → 基于实验反馈优化标准实验模板
6. **AI模型实验验证** → 在真实实验中验证和优化AI模型性能
